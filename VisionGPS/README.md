# 📡 VISION GPS — دليل بناء APK

## 🚀 الطريقة الأسهل: GitHub Actions (مجاني 100%)

### الخطوات:

**1. أنشئ حساباً على GitHub**
- اذهب إلى: https://github.com
- سجّل حساباً مجانياً

**2. أنشئ مستودعاً جديداً**
- اضغط زر "New repository"
- اسمه: `VisionGPS`
- اتركه Public أو Private
- اضغط "Create repository"

**3. ارفع ملفات هذا المجلد**
- اضغط "uploading an existing file"
- ارفع كل الملفات والمجلدات من هذا الـ ZIP
- اضغط "Commit changes"

**4. انتظر 3 دقائق**
- اذهب إلى تبويب "Actions"
- ستجد workflow يعمل باسم "Build VISION GPS APK"
- انتظر حتى يكتمل (✅ أخضر)

**5. حمّل الـ APK**
- اضغط على الـ workflow المكتمل
- في الأسفل ستجد "Artifacts"
- حمّل `VISION-GPS-release`

---

## 📱 تثبيت APK على هاتفك

1. انقل الملف إلى هاتفك
2. في إعدادات الهاتف: **الأمان → تثبيت من مصادر غير معروفة → تفعيل**
3. افتح ملف الـ APK وثبّته
